HLD interview structure
1. Clarify requirements (5 min) — functional + non-functional + scale estimates
2. High-level design (15 min) — components, data flow, API design
3. Deep dive (10 min) — DB schema, scaling bottlenecks, failure handling
4. Trade-off discussion — justify every choice, know the alternatives
